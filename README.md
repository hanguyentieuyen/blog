
# hayen.blog
hayen.blog is a web application built using NextJS, TypeScript and other technologies. This website is purposefully help me write about programming knowledge. 






## Tech Stack
- Typescript
- NextJS
- NextAuth
- TailwindCSS
- Prisma
- PostgreSQL
- Firebase


## Run Locally

Clone the project

```bash
  git clone https://github.com/hanguyentieuyen/hayen.note.git
```

Go to the project directory

```bash
  cd hayen.blog
```

Install dependencies

```bash
  yarn install
```

Start the server

```bash
  yarn dev
```


## What I learned
- Store images on firebase storage
- Authenticate users by OAuth provider: Github, Google 
- Use dynamic router on NextJS 
- Use Prisma ORM to process query data
